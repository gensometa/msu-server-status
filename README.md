# MSU Server Status

MapleStory Universe ゲームサーバーの稼働状況データ。

このリポジトリには `server-status.json` のみが含まれます。
VPS上のプローブスクリプトによって自動更新されます。

## GitHub Pages URL

```
https://gensometa.github.io/msu-server-status/server-status.json
```

## GitHub Pages 有効化

1. リポジトリの Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, / (root)
4. Save
